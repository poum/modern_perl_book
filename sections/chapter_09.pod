=head0 Gérer de vrais programmes

Z<writing_real_programs>

Un livre peut vous apprendre à écrire de petits programmes pour résoudre
de petits problèmes d'exemple. Vous pouvez apprendre plein de syntaxe de
cette façon. Pour écrire de vrais programmes qui résolvent de vrais problèmes,
vous devez aprendre à I<gérer> le code écrit dans votre langage. Comment
organisez-vous votre code ? Comment savez-vous qu'il fonctionne ? Comment
pouvez-vous le rendre robuste face aux erreurs ? Qu'est-ce qui rend votre
code concis, clair et maintenable ?

Perl moderne fournit de nombreux outils et techniques pour écrire de 
vrais programmes.
