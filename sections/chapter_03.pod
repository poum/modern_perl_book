=head0 Le langage Perl

Comme une langue, la globalité de Perl est une combinaison de plusieurs parties
plus petites mais interconnectées. Contrairement aux langues, où les nuances, 
le ton de la voix et l'intuition permettent aux gens de communiquer malgré de
légers malentendus et des concepts flous, les ordinateurs et le code source 
nécessitent de la précision. Vous pouvez écrire du code Perl efficace sans
connaître chaque détail de chaque fonctionnalité du langage, mais vous devez
comprendre comment elles fonctionnent ensemble pour bien écrire du code Perl.
